## Maximum Perimeter Triangle
🔗 [Hackerrank бодлого линк](https://www.hackerrank.com/challenges/maximum-perimeter-triangle)

### 🇲🇳 Монгол орчуулга  
Хамгийн их периметртэй гурвалжин  
Өгөгдсөн урттай гурав сонгон авч, гурвалжин үүсгэж болох хамгийн их периметрийг ол.

### ✅ Тайлбар  
Массивыг өсөхөөр эрэмбэлж, хамгийн их периметртэй гурвалжинг олохын тулд хамгийн том гурван утгаас эхлэн шалгана. Гурвалжны тэгшитгэлийг (a + b > c) шалгана.
